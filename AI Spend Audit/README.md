"AI Spend Audit Tool"

A web app that analyzes a company's AI tool usage and identifies potential cost savings by recommending better plans or alternatives.

Who it's for
Startup founders, developers, and teams using AI tools like ChatGPT, Claude, Copilot, etc.

Features
- Input AI tool usage
- Audit spending
- Show savings opportunities
- AI-generated summary
- Shareable results

 Quick Start
```bash
npm install
npm run dev
